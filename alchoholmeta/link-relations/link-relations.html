API Documentation

Link Relation Description URI
almeta:add-bar Add a bar to the collection /api/bars/
almeta:delete-bar Delete a bar from the collection /api/bars/<bar:bar>/
almeta:in-bar A link to the bar where the drink is sold /api/bars/<bar:bar>/
almeta:delete-tapdrink Delete a tapdrink from the collection /api/bars/<bar:bar>/tapdrinks/<drink_name>/<drink_size>/
almeta:add-tapdrink Add a tapdrink to the collection /api/bars/<bar:bar>/tapdrinks/
almeta:delete-cocktail Delete a cocktail from the collection /api/bars/<bar:bar>/cocktails/<cocktail_name>/
almeta:add-cocktail Add a cocktail to the collection /api/bars/<bar:bar>/
